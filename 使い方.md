# 使い方ガイド

弓木奈於さんの生写真の所有を管理するアプリの使い方です。はじめての方でも順番どおりに進めれば使えるように書いています。

運用の基本: **メイン＝iPhone**（バインダーを整理しながら○×をつける）／ **PC＝閲覧とバックアップ保管**

アプリ: https://nogi-photo-manager46.vercel.app

---

## 1. 初期セットアップ

### iPhone（メイン）

1. Safari でアプリのURLを開きます。
2. 画面下の共有ボタンを押し、ホーム画面に追加を選びます（以後はホーム画面のアイコンから起動でき、オフラインでも動きます）。
3. 前に使っていた所有データがあれば入れます。`yumiki_nao.ownership.json` を AirDrop などで iPhone のファイルアプリに保存し、アプリの 設定タブ → 読み込み（復元） → そのファイルを選ぶ、の順で復元します。
4. 設定タブ → データ保護を有効にします（ブラウザの自動削除からデータを守ります）。
5. 設定タブ → あなた で、ニックネームを入れておきます（後でトレードの共有ファイルに、持ち主の名前として入ります）。

### PC（閲覧用）

1. ブラウザでURLを開きます（ブックマークをおすすめします）。
2. iPhone と同じように 設定タブ → 読み込み（復元） で所有データを入れます。

## 2. 画面の見かた

- 画面下に3つのタブがあります。**コレクション**（バインダーとセットの一覧・○×をつける）／ **統計**（コンプ率やダブりの数）／ **設定**（バックアップ・ニックネーム・オーナー解除など）。
- コレクション画面の右上には3つのアイコンがあります。**カメラのマーク＝一括取込**（写真からまとめて登録）／ **左右の矢印マーク＝トレード** ／ **虫めがねマーク＝検索**。

## 3. 日常の操作（○×をつける）

| やりたいこと | 操作 |
|---|---|
| 新しく買った写真を登録 | 右上の虫めがねで検索 → セットを開く → 写真をタップして○にする |
| セットをまるごと買った | セット詳細の下部にある すべて所有にする を押す |
| 封入など1枚もの | 一覧の右端の○を直接タップ |
| ダブりを記録（2枚以上） | セット詳細で枚数を増やす（2枚以上はダブり扱いになります） |
| 次に買うものを探す | 検索画面で 未所有だけ をONにする |
| 進捗をながめる | コレクション画面のゲージ、または統計タブ |

## 4. カメラ・写真からのAI自動取込

写真を撮る、または選ぶだけで、どのセットのどのポーズかを自動で判別して登録できる機能です。手入力より速く登録できます。

1. コレクション画面の右上のカメラのマークを押して、取込画面を開きます。
2. どちらかを選びます。
   - カメラで撮る（連続）: その場で続けて撮影できます。撮った端から順番に解析が始まります。
   - 写真から選ぶ（複数OK・最大30枚）: 端末に保存済みの写真をまとめて選べます。
3. 少し待つと、写真ごとに 判別されたセットとポーズ枠 が表示されます。
   - 自信がある写真は、セットと枠が自動で割り当てられます。そのまま保存できます。
   - あいまいな写真は、候補が並びます。先頭の候補をタップすると確定でき、隣から他の候補も選べます。
4. 割り当てを確認したら、保存します。
5. コツ: 1枚の画像に何枚か並べて撮ると、少ない判定回数でたくさん取り込めます。

### 気をつける点

- ★レア（R・SR）の写真は、枠が要確認として仮に入るだけで自動確定はしません。必ず枠が合っているか自分で確認してください（見分け方は次の項目）。
- AIの判定は非オーナーの場合、**1日30回まで**です。取込画面に本日の残り回数が出ます。上限に達しても、手動でセットと枠を選んで保存でき、翌日にはまた自動判定が使えます。
- すぐに制限を外したいときは、設定タブ → オーナー解除 でパスワードを入れると無制限になります。

## 5. レア（R・SR）の登録

8種セットなどに含まれるレア写真は、見た目で通常版と見分けます。

- 通常版: 下の余白の文字がきれいな活字で、表面にキラキラ（ホログラム）がありません。
- R: 文字が手書き風で、表面にキラキラがあり、下の余白が白いものです。
- SR: 手書き風＋キラキラに加えて、下の余白やフチが虹色に光っているものです。

AIの自動取込ではR/SR枠を要確認として仮に入れるだけなので、上の見分け方で正しい枠か確認してから保存してください。

## 6. 「その他」= カタログにまだ無い写真

会場限定など、まだアプリのカタログに無い写真の仮置き場です。

1. コレクション画面で その他 のバインダーを開きます。
2. ＋未分類の写真を追加 から、写真を1枚ずつ足します（名前は任意ですが、付けておくと後で見つけやすくなります）。同じ名前がすでにあると重複の警告が出ます。
3. 後日その写真がカタログの正式なセットに加わり、付けた名前がセット名と一致すると、そのセットを開いたときに 本セットへ移動 の案内が出ます。押すと、画像と所有がまとめて正式なセットへ移り、その他の仮置き枠は消えます。

## 7. トレード

ダブりを譲ったり、欲しい写真を相手と交換したりするための機能です。サーバは使わず、ファイルを渡し合って突き合わせます。

- ダブり: 2枚以上持っている写真が、譲れる候補になります。
- ♡（特に欲しい）: 欲しい写真に♡を付けておくと、求リストに入ります。
- 求／譲リスト: X(旧Twitter)などに貼り付けられるテキストを作れます。
- 共有ファイルで突き合わせ: コレクション画面の右上（左右の矢印マーク）からトレード画面を開き、自分の共有ファイル（JSON）を書き出して相手に渡します。相手のファイルを読み込むと、相手からもらえる写真と、相手に渡せる写真が一覧になります。共有ファイルには譲れるものと特に欲しいものだけが入り、所有の全部は出ません。

## 8. バックアップ（とても大切）

○×や添付画像は **iPhoneの中にしか保存されていません**。ホーム画面からアプリを削除すると一緒に消えます。バックアップがあれば読み込みで元に戻せるので、習慣にしてください。最後の書き出しから14日たつと、ホーム画面で催促が出ます。

バックアップは2種類あり、別のファイルに分かれています。

- **○×データ（軽いJSON）**: 所有・枚数・♡などの記録です。設定タブ → 書き出し（バックアップ） から保存します。同じ名前で上書き保存するか、日時つきの新規保存にするかを選べます。
- **添付画像（大きいZIP）**: 実物の写真です。設定タブ → 画像をまとめて書き出す（ZIP） から保存します。枚数が多いと大きくなるので、○×データとは別建てにしています。

保存先は iCloud Drive（ファイルアプリ）がおすすめです。たまにPCにもコピーしておくと二重で安心です。

### 復元のしかた（順番が大切）

1. 先に 設定タブ → 読み込み（復元） で、○×データのJSONを読み込みます。
2. そのあと 設定タブ → 画像ZIPを読み込む で、画像を読み込みます。

必ずJSONを先に読み込んでください。先に枠（所有の情報）を戻してから画像を戻す順番です。

## 9. PC側を最新にしたいとき

自動では同期されません。iPhone で 書き出し → AirDrop で Mac に送る → PC のアプリで 読み込み、の順で最新にします。PCでは○×を付けず、閲覧専用にするのがおすすめです。

## 10. 新作の生写真が出たとき

- カタログへの正式な追加: カタログを更新して push すると、本番へ自動で反映されます。反映後にアプリを開き直すと新しいセットが増えます（**○×は消えません**）。手順の詳細は `README.md` のカタログの更新手順を参照してください。
- すぐ手元で記録したいとき: 上の「その他」に仮置きしておけます。後でカタログに正式追加されたら、本セットへ移動でまとめて移せます。
