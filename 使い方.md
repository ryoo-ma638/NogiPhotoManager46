# 使い方ガイド

運用方針: **メイン = iPhone**（バインダー整理しながら○×）／ **PC = 閲覧・バックアップ保管**

アプリ: https://nogi-photo-manager46.vercel.app

---

## 1. 初期セットアップ

### iPhone（メイン）
1. SafariでアプリのURLを開く
2. 共有ボタン →「**ホーム画面に追加**」（以後はホーム画面のアイコンから起動。オフラインでも動く）
3. 所有データを入れる: `yumiki_nao.ownership.json` をAirDrop等でiPhoneの「ファイル」に入れる → アプリの **設定 → 読み込み（復元）** → ファイルを選ぶ → 復元する
4. **設定 → データ保護を有効にする**（ブラウザの自動削除から守る）

### PC（閲覧用）
1. ブラウザでURLを開く（ブックマーク推奨）
2. 同じように **設定 → 読み込み（復元）** で所有データを入れる

## 2. 日常の操作（iPhone）

| やること | 操作 |
|---|---|
| 新しく買った写真を登録 | ホーム右上の🔍で検索 → セットを開く → 写真をタップで○ |
| セットまるごと買った | セット詳細の下部「**すべて所有にする**」 |
| 封入（1枚もの） | 一覧の右端の○を直接タップ |
| 次に買うものを探す | 検索画面で「**未所有だけ**」をON |
| カタログにないセット（会場限定など） | バインダー画面右上の **＋** で追加（枠は後から編集可） |
| 進捗を眺める | ホームのゲージ／統計タブ |

## 3. バックアップ（重要・月1回目安）

**○×のデータはiPhoneの中にしかない**ので、これだけは習慣に：

1. iPhoneで **設定 → 書き出し（バックアップ）** → 「ファイル」の **iCloud Drive** に保存
2. たまにPCにもコピーしておくと二重で安心

⚠️ **ホーム画面からアプリを削除するとデータも消えます**。消してしまっても、バックアップファイルがあれば「読み込み」で完全復元できます。

## 4. PC側を最新にしたいとき

iPhoneで **書き出し** → AirDropでMacへ → PCのアプリで **読み込み**。
（自動同期はされないため、PCで○×は付けず閲覧専用にするのがおすすめ）

## 5. 新作の生写真が出たら

方法A（おすすめ）: 開発チャットで「〇〇（セット名・何種類か・✓か◦か）が出た」と伝える → カタログ更新 → アプリを開き直すと反映（**○×は消えません**）

方法B: アプリのバインダー画面「＋」で自分で追加（後でカタログに正式追加したら手動分は削除すればOK）
